# Dungeon Raid — Roguelite

Phaser 3 + Vite で構築した一筆書き型ローグライトゲームです。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開くとゲームが起動します。

## 遊び方

1. **🧙（プレイヤー）をクリック**して経路の起点にする
2. 隣接マスをクリック or ドラッグして経路を引く（縦横斜め可、Uターン不可）
3. **終着点は空マス**である必要あり
4. **Enter / ✓ボタン**で移動確定
5. Escape / ✗ボタンでキャンセル

## ゲームルール

- **ターン順**: 敵フェーズ（スポーン＋スキル） → プレイヤーフェーズ
- **コンボ**: 1ターンで複数の敵を通過すると攻撃力にボーナス（×1.2, ×1.4…）
- **コイン10枚**: ショップが開き、4択アップグレード
- **先制攻撃**: プレイヤーの攻撃が先に当たり、敵HPが0になれば反撃なし

## アップグレード

| 名前 | 効果 |
|------|------|
| ⚔ ATK強化 | 攻撃力 +1 |
| 👟 歩数強化 | 最大歩数 +1 |
| 🩸 ドレイン | 敵撃破でHP+1 |
| 💣 ボム | 移動終点の周囲8マスにATKダメージ |
| ❤ MaxHP強化 | MaxHP +5 & HP回復 |
